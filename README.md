❗If termux packages are not installed, 
you need to install termux pakets.

you are a member to other groups. 

---

❗you need api id and api hash,
you can get api id and api hash by registering at 

[api id-hash](https://my.telegram.org.)

`
API_ID=XXXXXXXXXX,
API_HASH=XXXXXXXXXX
API_PHONE_NUMBER=XXXXXXXXXX
`

To install ngrok, you need to register from ngrok site, 
copy the authtoken, paste it into terminal, 
then type `/ngrok http 80' and it will work, ... 
the port must be open for ngrok to work .
